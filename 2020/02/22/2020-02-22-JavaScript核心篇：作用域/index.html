JavaScript 作用域

JavaScript是採用語法作用域，屬於靜態作用域的，靜態作用域是變數的作用域在語法解析的時候，就已經確定它的作用域了，確定完就無法再改變。

如果作用域裡找不到變數的時候，就會向外查找，找不到的時候就會報錯(not define)。
