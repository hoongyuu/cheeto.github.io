什麼是函式

結果會呈現 ↓

function 有預設一個回傳的值，所以它能夠被當作表達式來用。


當我們使用 function 去宣告一個字詞的時候，這個字詞就會變成一段函式，它就能夠有下列三種特性。

函式的名稱是選用的，意思就是可以有名稱也可以沒有，有名稱的稱為「具名函式」；沒名稱的稱為「匿名函式」。

函式表達式也能用具名函式

用函式表達式也能夠使用具名函式，但是只能在函式內被調用。
