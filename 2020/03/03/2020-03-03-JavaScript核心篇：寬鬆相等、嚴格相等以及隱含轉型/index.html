寬鬆相等、嚴格相等以及隱含轉型

「== 允許在比較中強制轉型，=== 不允許。」

嚴格相等

嚴格相等必須值、型別都要相同才會回傳 true。

寬鬆相等

寬鬆等於只要值相等就會回傳 true，這裡說的值包含字串、布林轉化過來的值。

Null 跟 undefined 不能被轉化為數字型別。

物件與非物件比對

在物件與非物件比對的時候，是使用包裹物件來進行轉換。

物件比對

物件經過隱含轉型後，會變成 [object Object]。

object 指向的是記憶體位置，所以 a == b 時不會相等。
