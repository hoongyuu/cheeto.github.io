Truthy Falsy

在 JavaScript 中，truthy(真值) 指的是在布林值判斷中，轉換後的值為真的值。我們也可以另外一個方向來理解「除了 falsy 以外的值都是 truthy」，那我們就可以來看看 falsy 的值有哪些了。如下

  • 0
  • NaN
  • ‘’ (空字串)
  • false
  • null
  • undefined
