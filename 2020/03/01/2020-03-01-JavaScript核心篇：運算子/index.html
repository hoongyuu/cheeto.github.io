運算子

運算子一定會回傳一個結果，JavaScript 常見的運算子有二元運算子、一元運算子，以及一種特殊的 三元運算子，也就是 條件運算子。

一元運算子

一元運算子是只需要一個運算元的運算。

delete

delete 就算是一元運算子。可以用來刪除物件、物件的屬性，或者在陣列中選取索引指定刪除的目標。語法如下

typeof

typeof 也算是一元運算子。

條件運算子 是 JavaScript 唯一的三元運算子。
