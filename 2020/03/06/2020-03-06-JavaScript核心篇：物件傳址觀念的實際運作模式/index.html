第一步驟

因為物件型別是傳址的特性，所以 var b = a; 時是指向同一個位址。

第二步驟

雖然 =(等號) 的相依性是由右至左，但a.y = a = {x: 2};是同時運行的，所以 a.y 和 a 都會指向同一個位址。
