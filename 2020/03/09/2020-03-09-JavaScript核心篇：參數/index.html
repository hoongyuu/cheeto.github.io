物件參數

在函式參數當中如果是物件的話，還是會維持原本傳址的特性，所以 family 才會跟著被變動了。

callback function

  • 洋紅色: functionB 執行時將函式作為參數傳入 fn，並在 functionB 內執行
  • 綠色: fn 的小明作為參數傳遞給 functionB 執行時的函式
